# Impulsfp

## Descripció
Aquest projecte és una aplicació feta amb C# a Visual Studio. Consisteix en un sistema senzill d’inici de sessió on l’usuari ha d’introduir el seu nom i contrasenya.

Si les dades coincideixen amb les que hi ha a la base de dades, es permet l’accés a l’aplicació. En cas contrari, no es pot iniciar sessió.

## Funcionalitats
- Pantalla d’inici de sessió
- Validació d’usuari i contrasenya amb la base de dades
- Accés a un formulari segons el rol identificat
- Opció de tancar sessió i tornar a la pantalla inicial

## Tecnologies
El projecte està desenvolupat amb:
- C#
- Visual Studio
- Windows Forms

## Com utilitzar-lo
1. Obrir el projecte amb Visual Studio
2. Executar l’aplicació
3. Introduir el nom d’usuari i la contrasenya
4. Si les dades són correctes, s’obrirà el formulari segons el rol
5. Des d’allà es pot tancar la sessió i tornar a l’inici

## Estat del projecte
El projecte està en desenvolupament. De moment només funciona el sistema de login i el formulari principal encara no està complet.
